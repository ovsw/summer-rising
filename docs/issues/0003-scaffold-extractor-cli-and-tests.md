- [ ] Test runner is configured and can run a placeholder fixture-based test.
- [ ] CLI command names and arguments use glossary terms from CONTEXT.md.
- [ ] Required configuration fails loudly when absent; no dummy fallbacks are added for required values.
- [ ] No node_modules files are edited directly.
